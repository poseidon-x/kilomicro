Savings & Loans ››

After the loan application has been entered into the system, it is then forwarded to the manager by the cashier  to be check listed whether all supporting documents have been submitted by the client.If all supporting documents have been submitted, the loan can then be approved





After Manage Loans has been selected, find the name of client by typing either the Surname, Other Names or Account No. and click on Find Client. Related search results would be displayed then click on the expand arrow of the appropriate client to display details. Select the figure below Manage Check List to retrieve all details to be able to check if all documents needed have been submitted by client as shown in the images below.



After selecting the client as shown above, tick the appropriate box provided to indicate if the client has submitted all documents needed. Comments can also be made if necessary Please remember to save checklist for Loan Application when done.
